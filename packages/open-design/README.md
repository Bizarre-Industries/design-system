# @bizarre/open-design

Private, deterministic Bizarre Industries brand and interface working manual generated from the canonical repository packages. It is nonpublishable while the owner-selected Continuous Lens Aperture v2 remains governed-provisional.

The generated bundle contains the exact Astronomical Atlas authority, canonical token and UI outputs, approved governed assets, an offline preview, the portable UI fixture, a manifest-derived token tree, review HTML, and an evidence report. Provisional Atlas SVGs are isolated below `generated/provisional/`, retain their nonpublishable manifest, and never enter `generated/assets/`. The bundle intentionally excludes the proposal's compiled CSS, generated mockups, and PDF.

`generated/manual/print.html` is review source only until the aperture receives explicit master approval and a selected renderer proves that the governed fonts are embedded and a rendered-page raster comparison passes. Physical production gates remain `NOT VERIFIED` until real samples or calibrated display evidence are recorded.
